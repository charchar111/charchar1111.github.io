## 컨셉

이 프로젝트는 카카오톡의 UI를 소재로 아래 주제에 대해 연구하였습니다.

- [재사용에 적합한 컴포넌트 디자인](#컴포넌트-디자인)
- BEM을 통한 가독성 및 유지보수-확장성(#BEM)
- 반응형 디자인과 적응형 디자인(#반응형 디자인-적응형 디자인)
- 디자인 부서와 협업에 용이한 코드 작성(#반응형 디자인을 위한 단위 문제)

## 기능
- 카카오톡 앱과 유사한 UI의 페이지
    - 로그인(index.html)
    - 친구 리스트(friends.html)
    - 채팅 리스트(chats.html)
    - 채팅창(chat.html)
    - 컨텐츠 더보기(find.html)
    - 기타 컨텐츠(more.html)
    - 세팅(settings.html)

 - 주요 컴포넌트의 상호작용 애니메이션
   - ex: 버튼 클릭
